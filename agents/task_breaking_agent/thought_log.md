# Task Breaking Agent - Thought Log

## Agent Config
- **Role:** Decompose approved requirements into sprint tasks with dependencies, estimates, and agent assignments
- **Upstream:** Root_Requirement_Agent
- **Downstream:** Master_Orchestrator, Progress_Monitor, stack Dev Leads

---

## Skill Set
1. **Requirement Decomposition** -- Break 1 TRD spec into N atomic, single-agent tasks
2. **Sprint Sequencing** -- Order tasks respecting dependency graph, backend-before-frontend within sprints
3. **Complexity Estimation** -- S/M/L sizing, XL must be split further
4. **Parallelism Detection** -- Identify concurrent task groups per sprint
5. **Cross-Stack Flagging** -- Mark tasks needing Asynchronous Handshake coordination
6. **Scope Change Handling** -- Diff requirements, re-estimate affected sprints on PRD changes
